## Overview
Added comprehensive agent filtering feature to PAX Purview Audit Log Processor. New -AgentId and -AgentOnly switches allow filtering Copilot usage records by AgentId field. Includes intelligent progress bar allocation (90% pre-parsing, 10% filtering), agent metrics in final summary, and conditional display of summary sections (parallel execution, explosion metrics, performance optimization). Fixed progress bar accuracy and made summary output more concise by removing verbose CSV column listings.
