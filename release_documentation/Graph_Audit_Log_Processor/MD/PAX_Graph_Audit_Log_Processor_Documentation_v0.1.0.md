## 🎯 Overview

The **PAX Graph API Audit Log Processor** extracts Copilot usage analytics and M365 application metrics via the Microsoft Graph API. This script is optimized for high-level reporting, usage analytics, and Power BI dashboard integration.

**This script is currently in development.** Check back soon for the initial release!

---

## 📋 Planned Features

### **Report Types**

- **Copilot Usage Reports** - User-level and tenant-level adoption analytics
- **M365 App Usage** - Teams, Outlook, Word, Excel, PowerPoint Copilot metrics
- **License Tracking** - Copilot license assignment and utilization
- **Activity Summaries** - Aggregated usage patterns and trends

### **Key Capabilities**

- ✅ Server-side aggregation for fast retrieval
- ✅ Power BI-ready output formats
- ✅ Scheduled execution support
- ✅ Multiple authentication methods
- ✅ CSV and JSON export options
- ✅ Incremental data updates
